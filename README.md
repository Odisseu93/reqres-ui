# ReqResUI

ReqResUI is a user interface built using Next.js, Tailwind CSS, and TypeScript. It provides a sleek and intuitive interface to interact with the [ReqRes API](https://reqres.in/). Explore user management functionalities, such as creating, updating, and deleting users. View and manage user profiles effortlessly. ReqResUI simplifies the process of interacting with the ReqRes API, making it easy to work with user data


### Teach Stack 
- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Typesscript](https://www.typescriptlang.org/)
- [Tailwind](https://tailwindcss.com/)
- [Headless UI](https://headlessui.com/react/dialog)
- [Font Awesome](https://fontawesome.com/)
- [ReqRes API](https://reqres.in/)
